
# yt-downloader

A simple yt downloader made with python using pytube


## Features

- Audio Downloader
- Video Downloader
- Discord Rich presence (Coming soon...)
- Cross platform


## Installation

Install requirements

```bash
  pip install -r requirements.txt
```
    
## Usage/Examples

```javascript
python3 vid.py https://www.youtube.com/watch?v=XYlI5iyWP2U
```
